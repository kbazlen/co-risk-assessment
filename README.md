# Overview
Repository for analsis for the Colorado Climate Resilience Roadmap. 

# Getting Started
1. from home dir, `git clone git@github.com:kbazlen/co-risk-assesment.git`
2. Create a conda env: `conda create --name 2026_nvhackathon python=3.13.2`
3. Run the command: `conda activate co_rm`
4. cd into the repo cloned in 1.
5. prior to running pip below, make sure the branch is correct (install from main!)
6. Install the repo and its dependencies by running: `pip install -e .` (or `pip install -e .[ml]` to also install ML stack, see .toml)
